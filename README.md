# 📚 Book API

A simple RESTful API built with Node.js and Express.js to manage a list of books. This project supports Create, Read, Update, and Delete (CRUD) operations with data stored in memory.

---

## 🔧 Features

- ➕ Add a book (POST)
- 📖 View all books (GET)
- ✏️ Update a book (PUT)
- ❌ Delete a book (DELETE)

---

## 🚀 How to Run Locally

```bash
npm install
node index.js
